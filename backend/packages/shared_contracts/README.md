# Shared Contracts

Контракты межсервисного взаимодействия.

## Назначение

Этот пакет содержит:
- События (events) для асинхронного взаимодействия
- Команды (commands) для синхронных вызовов
- Схемы сообщений (message schemas)
- Контракты API между сервисами

## Использование

```typescript
import { DraftCreatedEvent, CommitDraftCommand } from '@site-builder/shared-contracts';
```

## Структура

```
shared_contracts/
├─ src/
│  ├─ events/             # События (domain events)
│  ├─ commands/           # Команды
│  ├─ queries/            # Запросы
│  └─ schemas/            # Схемы сообщений (JSON Schema, Avro и т.д.)
└─ package.json
```

## Правила

1. **Версионирование**: Контракты должны быть версионированы
2. **Обратная совместимость**: Старайтесь сохранять обратную совместимость
3. **Документация**: Каждый контракт должен быть задокументирован
