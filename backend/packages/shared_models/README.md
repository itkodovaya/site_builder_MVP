# Shared Models

Общие доменные модели и инварианты, используемые между сервисами.

## Назначение

Этот пакет содержит общие доменные модели, которые:
- Используются несколькими сервисами
- Не содержат специфичной для сервиса логики
- Определяют общие инварианты и правила

## Использование

```typescript
import { SiteConfigModel } from '@site-builder/shared-models';
```

## Структура

```
shared_models/
├─ src/
│  ├─ site/              # Модели сайта
│  ├─ validation/         # Общие правила валидации
│  └─ invariants/         # Бизнес-инварианты
└─ package.json
```

## Правила

1. **Только общие модели**: Не выносите модели, специфичные для одного сервиса
2. **Без логики сервисов**: Модели не должны содержать логику конкретных сервисов
3. **Инварианты**: Общие бизнес-правила и инварианты
