# Shared Types

Общие типы, DTO и контракты, используемые между сервисами.

## Назначение

Этот пакет содержит:
- Общие типы данных (SiteConfig, Draft, PublishedSite и т.д.)
- DTO для межсервисного взаимодействия
- Схемы валидации (Zod)
- Версии схем и миграции типов

## Использование

```typescript
import { SiteConfig, DraftDTO } from '@site-builder/shared-types';
```

## Структура

```
shared_types/
├─ src/
│  ├─ site/              # Типы конфигурации сайта
│  ├─ draft/              # Типы черновиков
│  ├─ events/             # Типы событий
│  ├─ errors/             # Типы ошибок
│  └─ schemas/            # Zod-схемы для валидации
└─ package.json
```

## Правила

1. **Независимость от сервисов**: Типы не должны зависеть от логики конкретных сервисов
2. **Версионирование**: При breaking changes создавайте новые версии схем
3. **Валидация**: Используйте Zod для валидации на границах сервисов
