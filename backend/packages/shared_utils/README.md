# Shared Utils

Общие утилиты, используемые между сервисами.

## Назначение

Этот пакет содержит:
- Логирование (logger)
- Валидация (helpers для валидации)
- Утилиты времени (time helpers)
- Генераторы ID
- Result/Error типы и утилиты

## Использование

```typescript
import { logger, generateId, Result } from '@site-builder/shared-utils';
```

## Структура

```
shared_utils/
├─ src/
│  ├─ logger/            # Логирование
│  ├─ validation/         # Утилиты валидации
│  ├─ time/               # Утилиты времени
│  ├─ id/                 # Генераторы ID
│  ├─ result/             # Result/Error паттерн
│  └─ errors/             # Общие ошибки
└─ package.json
```

## Правила

1. **Без зависимостей от сервисов**: Утилиты должны быть независимы
2. **Переиспользуемость**: Только действительно общие утилиты
3. **Производительность**: Утилиты должны быть эффективными
