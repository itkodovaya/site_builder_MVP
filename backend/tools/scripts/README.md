# Development Scripts

CLI-скрипты для разработки.

## Назначение

Утилиты для:
- Создания новых сервисов (scaffold)
- Проверки правил структуры
- Генерации кода
- Миграций

## Доступные скрипты

- `scaffold-service.sh` - Создание нового сервиса с базовой структурой
- `check-structure.sh` - Проверка соответствия структуры правилам
- `generate-types.sh` - Генерация типов из схем

## Использование

```bash
# Создание нового сервиса
./tools/scripts/scaffold-service.sh my-service

# Проверка структуры
./tools/scripts/check-structure.sh
```
