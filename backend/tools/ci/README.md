# CI/CD Helpers

Хелперы для CI/CD пайплайнов.

## Назначение

Скрипты и конфигурации для:
- Линтинга и форматирования
- Тестирования
- Сборки
- Деплоя

## Структура

```
ci/
├─ lint.sh               # Скрипт линтинга
├─ test.sh               # Скрипт тестирования
├─ build.sh              # Скрипт сборки
└─ deploy.sh             # Скрипт деплоя
```
